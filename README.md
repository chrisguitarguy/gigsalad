# Gigsalad Test Project

## Requirements

- PHP 5.6+
- Composer
- NodeJS & NPM

## Getting Started

#### 1. Clone the Repo

```
git clone https://github.com/chrisguitarguy/gigsalad.git
cd gigsalad
```

#### 2. Install the Fixtures, Depencencies, Build the Frontend Code, & Run the Server

```
make all
```
