# Gigsalad Test Project

## Requirements

- PHP 5.6+
- Composer
- NodeJS & NPM

## Getting Started

#### 1. Clone the Repo

```
git clone https://github.com/chrisguitarguy/gigsalad.git
cd gigsalad
```

#### 2. Install the Depencencies

```
make deps
```

#### 3. Build the Frontend Code

```
make build
```

#### 4. Run the Dev Server & Visit `localhost:9090`

```
make serve
```
